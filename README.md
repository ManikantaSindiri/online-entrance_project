http://localhost:3000                                                                             # Online Entrance Registration System 

A simple web-based student registration project developed using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL.

This project allows students to register through an online entrance form and stores their details in a MySQL database.

---

# Project Features

- Student Registration Form
- MySQL Database Integration
- Store Student Details
- View Registered Students
- Responsive UI Design
- Simple Frontend and Backend Integration

---

# Technologies Used

## Frontend
- HTML
- CSS
- JavaScript

## Backend
- Node.js
- Express.js

## Database
- MySQL
